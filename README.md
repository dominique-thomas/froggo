**A Cozy Virtual Pet Experiment**
Frogo is a small, Tamagotchi-inspired virtual pet experience featuring a single frog companion.

## Features
- Single frog companion with animated reactions  
- Pet, feed, play, and water interactions  
- Real-world day/night cycle with sleep behavior  
- Rare, context-based cosmetic hats (special days & random moments)  
- Persistent state (Frogo remembers affection across sessions)  
- Designed as a lightweight, cozy experience  

## Purpose
This project was created as a small experimental build focused on interaction design, state management, and emotional feedback through minimal UI.

## Technologies Used
- Unity (WebGL)  
- C#  
- HTML / CSS
